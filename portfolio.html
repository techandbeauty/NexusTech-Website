Tech Deck App

Tech Deck started as a simple flashcard study tool and evolved into a comprehensive learning platform featuring TARA (Teaching & Retention Assistant), an AI companion designed to help users learn and retain technical concepts.

The app includes multiple study modes, progress tracking, voice synthesis for pronunciation, and smart review algorithms based on spaced repetition.

Key Features

  • AI-powered tutoring assistant (TARA)
  • Flashcard study system
  • Multiple activation methods
  • Voice synthesis for learning
  • Progress tracking and analytics
  • Spaced repetition algorithm

Technologies Used

Swift, SwiftUI, OpenAI API, Firebase, Core Data, AVFoundation

Project Details

  • Educational Platform
  • AI Companion
  • Voice Integration
  • Study Analytics
